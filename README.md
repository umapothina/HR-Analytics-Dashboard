# HR Analytics Dashboard

## Project Overview
Developed an HR Analytics Dashboard using Power BI and SQL to analyze employee attrition, salary trends,and performance metrics.

## Features
- Employee attrition analysis
- Department-wise insights
- Salary trend analysis
- Performance visualization
- Interactive KPI cards and slicers

## Technologies Used
- Power BI
- SQL
- Excel

## Business Insights
- Identified departments with higher attrition
- Analyzed salary distribution across job roles
- Visualized employee performance metrics
